Remotion (React) video generator that produces product showcase/demo videos. Each project lives under `public/<id>/` with a `config.json` and `screenshots/` directory.

**Why:** User builds demo videos for client products (e.g. Stammer.ai, Scoutside). Fully config-driven so non-engineers can update.

**How to apply:** When touching video logic, always check `src/types/project.ts` for the schema, `src/lib/constants.ts` for timing constants, and `scripts/discover.ts` for how screenshots are auto-paired to features by index order.

Key design: `src/projects.config.ts` is AUTO-GENERATED by `scripts/sync-config.ts` — never edit it directly. Edit `public/<id>/config.json` then run `npm run sync`.

ScrollPan + CaptionOverlay use a shared scroll timeline (`src/lib/scrollTimeline.ts`) — scroll pauses at each caption's `atProgress` scroll position so sections are fully visible when captions appear.
